GS1 Barcode Syntax Engine - Android Mobile App
==============================================

This directory contains an Android Studio project (Kotlin) demonstrating mobile integration of the GS1 Barcode Syntax Engine with ML Kit barcode scanning.

Quick Start guide: <https://gs1.github.io/gs1-syntax-engine/java/org/gs1/gs1encoders/package-summary.html#android-mobile-app-quick-start-heading>
