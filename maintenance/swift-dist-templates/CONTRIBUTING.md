# Contributing

This repository is a **generated distribution** of the GS1 Barcode Syntax Engine Swift
binding. Its contents are produced from
[gs1/gs1-syntax-engine](https://github.com/gs1/gs1-syntax-engine) (`src/swift` plus the
vendored C library) and must not be edited here — changes would be overwritten on the next
sync.

Please file issues and open pull requests against
[gs1/gs1-syntax-engine](https://github.com/gs1/gs1-syntax-engine).
